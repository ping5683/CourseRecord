# 🧪 端到端测试完整指南

## 🚀 环境准备

### 第1步：选择数据库

#### 选项A：MySQL（推荐）
1. 确保MySQL已安装并运行
2. 创建数据库：`CREATE DATABASE course_management;`
3. 更新 `backend/.env` 中的数据库密码
4. 运行：`deployment-test/backend/start-mysql.bat`

#### 选项B：SQLite
1. 安装C编译器（TDM-GCC）
2. 运行：`deployment-test/backend/start-sqlite.bat`

### 第2步：启动前端

```bash
cd frontend
npm install
npm run dev
```

## 📱 完整测试流程

### 用户认证测试
1. 访问 http://localhost:3000
2. 注册新账号：用户名 `testuser`，密码 `123456`
3. 登录验证

### 课程管理测试
1. 点击"添加课程"
2. 填写课程信息：
   - 课程名称：`健身私教课`
   - 总金额：`3000`
   - 正式课时：`20`
   - 赠送课时：`5`
3. 保存并验证显示正确

### 课程安排测试
1. 进入课程详情页
2. 添加课程安排：
   - 星期：星期一、星期三、星期五

3. 保存设置

### 出勤功能测试
1. 返回主页
2. 找到课程，点击"签到"
3. 验证课时从25减少到24
4. 检查出勤记录

### 通知功能测试
1. 允许浏览器通知权限
2. 在设置中点击"测试通知"
3. 验证收到桌面通知

## 🆘 常见问题

### 数据库连接失败
- 检查MySQL服务状态
- 验证.env中的数据库配置
- 确认数据库已创建

### 前端无法访问后端
- 确认后端已启动
- 检查端口是否被占用
- 验证CORS配置

### 通知不工作
- 检查浏览器通知权限
- 确保使用HTTPS或localhost
- 刷新页面重试
